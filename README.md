# Ultimate Weekly Schedule

An interactive **Weekly Schedule** web page where you can view, edit, and export schedules for group members. Fully editable, with PDF, Excel, and DOCX export options, plus dark mode support.

## Live Demo

You can view the schedule live using GitHub Pages:

[Open Ultimate Weekly Schedule](https://dawitbt.github.io/weekly-schedule/)

## Features

- Editable table of members, locations, and times.
- Dark mode toggle for comfortable viewing.
- Download schedule as:
  - **PDF** (via browser print)
  - **Excel** (`.xlsx`)
  - **Word DOCX** (`.docx`)
- Save and load schedule as **JSON** for backup and sharing.
- Print-friendly layout.
